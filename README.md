# S3 Signer
Simple one-endpoint server to sign Amazon S3 File Upload policies so you can upload
files to S3 using browser-side technologies.

# Usage with Docker + Docker-compose
```shell
docker-compose up
```


# Usage with locally installed Node.js
```shell
cd app && npm start
```
